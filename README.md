## 🚀 Getting Started

### 🔧 Installation

1. Clone the Philosophers repository:
```sh
git clone --recursive https://github.com/San-tito/Philosophers
```

2. Change to the project directory:
```sh
cd Philosophers
```

3. Compile the dependencies:
```sh
make
```

### 🤖 Running Minishell
```sh
./philosophers
```
